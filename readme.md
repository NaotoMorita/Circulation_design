Wndmize
===========

WindmizeはLagrangeの未定乗数法を用いて、存在する桁にたいして指定したたわみを実現する抗力最小の循環分布(揚力分布)を求めるアプリケーションです

使い方やその他情報は私のブログ
http://d.hatena.ne.jp/yuukivel/
を参照して下さい。このサイトでは.exeファイルも公開しています。
